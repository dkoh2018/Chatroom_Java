# Java ChatRoom Project

## Prerequisites

- JDK 17+
- Maven 3.x

## Steps

1. Navigate to the project root:

   ```bash
   cd path/to/ChatRoom_Java
   ```

2. Build the project:

   ```bash
   mvn clean package
   ```

3. Run the application:
   ```bash
   java -jar target/chatapplication-1.0-SNAPSHOT.jar
   ```
